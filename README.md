<h1>🧩 LassoSS</h1>

<h2>A Chrome extension for freeform (lasso) screenshot cropping.</h2>

<h3>🚀 How to Use</h3>
Download the project and open the dist folder
Go to chrome://extensions/ in Chrome
Enable Developer Mode
Click Load Unpacked
Select the dist folder
Open any webpage
Click the LassoSS extension icon
Take a screenshot and draw a shape around the area
Close the shape → image will be downloaded automatically
<h3>📌 Note</h3>
Works only on the active tab
No data is sent anywhere (fully local)
